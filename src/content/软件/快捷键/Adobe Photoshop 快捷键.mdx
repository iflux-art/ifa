#### 工具切换技巧

- **循环切换工具**：按住 Shift + 工具快捷键，可循环切换同组工具

- **临时切换抓手工具**：按住空格键可临时使用抓手工具

## 实用技巧

### 快速调整画笔

- **调整画笔大小**：在画布上按住 Alt + 右键拖动（Windows）/ Option + 左键拖动（Mac）
- **调整画笔大小和硬度**：Alt + Ctrl + 左键拖动水平（大小）垂直（硬度）

### 快速填充

- **快速填充前景色**：Alt + Delete / Option + Delete
- **快速填充背景色**：Ctrl + Delete / Cmd + Delete

### 快速查看

- **临时切换到缩放工具**：按住 Ctrl / Cmd 并点击
- **临时切换到缩小工具**：按住 Alt / Option 并点击

### 图层蒙版

- **查看蒙版效果**：按住 Alt / Option 并点击蒙版
- **停用/启用蒙版**：Shift + 点击蒙版

### 快速复制

- **快速复制图层**：选中图层按 Ctrl + J / Cmd + J
- **拖动复制**：按住 Alt / Option 拖动图层
